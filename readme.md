## Requisitos para iniciar el app
- Ionic v7.2.0
- Ng v18.2.5
- npm v10.7.0

### Clonar el repositorio
` git clone https://github.com/a-rodriguez-c/ux-mobile-app` 

### Instalar dependencias
`npm install`

### Ejecutar la app
`ionic serve`
